# Contract Analysis & Risk Assessment Bot

A GenAI-powered legal assistant for Indian SMEs that analyzes contracts,
identifies risky clauses, and explains them in plain business language.

## Features
- Contract upload (PDF, DOCX, TXT)
- Clause extraction
- Risk scoring (Low / Medium / High)
- Plain-language explanations
- Audit logging
- Streamlit-based UI

## Tech Stack
- Python
- spaCy
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
